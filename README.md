# Katha Studio

Katha Studio is a React + Vercel app that turns curated Indian folk and regional stories into a 7-reel cinematic pack.

Current MVP includes:
- Agent 1 research dossier from curated source library
- Agent 2 seven-part story blueprint and script pack
- Agent 3 image, voiceover, music, and subtitle asset generation
- Agent 4 browser-side reel rendering and ZIP export

To run locally once Node is available:

```bash
npm install
npm run dev
```
